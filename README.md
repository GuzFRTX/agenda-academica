# Agenda Acadêmica

## Sobre o projeto

A Agenda Acadêmica é uma aplicação web desenvolvida com o objetivo de organizar a rotina de estudos de forma simples, eficiente e intuitiva.

O projeto foi concebido inicialmente como uma solução pessoal e evoluiu para uma aplicação personalizada, adaptada às necessidades específicas de uma cliente. A proposta é centralizar informações acadêmicas em um único ambiente, facilitando o planejamento e o acompanhamento das atividades.

## Objetivo

O principal objetivo da aplicação é auxiliar na organização da vida acadêmica, permitindo que o usuário tenha controle claro sobre suas tarefas, prazos e compromissos.

## Funcionalidades

- Cadastro de disciplinas
- Criação de tarefas vinculadas às disciplinas
- Definição de prazos para atividades
- Marcação de tarefas como concluídas
- Visualização organizada das atividades
- Interface simples e intuitiva

## Tecnologias utilizadas

- JavaScript
- HTML
- CSS
- Git e GitHub

## Estrutura do projeto

O projeto foi desenvolvido com foco em simplicidade e organização, priorizando uma experiência de uso fluida e acessível.

## Status do projeto

O projeto encontra-se em fase avançada de desenvolvimento, com a maior parte das funcionalidades principais já implementadas.

## Melhorias futuras
